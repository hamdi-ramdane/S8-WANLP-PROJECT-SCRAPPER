## Installation

```
git clone https://github.com/hamdi-ramdane/S8-WANLP-PROJECT-SCRAPPER.git
cd S8-WANLP-PROJECT-SCRAPPER
```

### On Windows

```
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt

```

### On Linux

```
python3 -m venv .venv
source myenv/bin/activate
pip install -r requirements.txt
```

## Running

```
python run.py
```
